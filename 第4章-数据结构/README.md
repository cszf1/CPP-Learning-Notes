## 1. 线性表

> 🔨 待补充详细内容和代码示例

### 概念
- 线性表是最基本、最常用的一种数据结构
- 由n个相同数据类型的数据元素组成的有限序列

### 特点
- 表中第一个元素只有一个后继
- 表中最后一个元素只有一个前驱
- 除首尾元素外，其他元素既有前驱也有后继
