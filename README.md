## 学习方法

1. 每章先看“本章目标”，知道自己要解决什么问题。
2. 代码一定要敲。只看代码就像看别人健身，挺热闹，但肌肉不会长在自己身上。
3. 遇到指针、引用、构造函数这些概念，先用类比理解，再回到代码验证。
4. 小题比大项目更适合入门。先写 30 行能跑的，再挑战 300 行。

## 编译运行

```bash
g++ -std=c++17 main.cpp -o main
./main
```

如果在 Windows 的 PowerShell 里运行，通常是：

```powershell
g++ -std=c++17 main.cpp -o main.exe
.\main.exe
```

## 最后一句

C++ 不难在“会写 hello world”，难在它很诚实：内存、对象生命周期、类型关系都摆在你面前。别怕，慢慢拆。大多数难点不是怪物，只是说明书写得太像说明书。
