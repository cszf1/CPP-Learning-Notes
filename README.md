# C++ 学习笔记

## 使用方法

- 第一遍：看概念和短代码，先理解“这东西干嘛用”。
- 第二遍：自己敲例子，改一两个地方，看程序怎么变。
- 第三遍：做题时回查“常见坑”。很多 bug 不是你笨，是 C++ 太会藏小纸条。

## 编译运行

```bash
g++ -std=c++17 main.cpp -o main
./main
```

Windows PowerShell：

```powershell
g++ -std=c++17 main.cpp -o main.exe
.\main.exe
```
