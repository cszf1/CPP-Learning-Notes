## 1. C++ 程序基本结构

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello C++" << endl;
    return 0;
}
```

- `#include` 引入头文件。
- `main` 是程序入口。
- `cout` 输出，`cin` 输入。
- `return 0` 表示程序正常结束。

## 2. 标识符、关键字与命名

标识符是变量名、函数名、类名等。命名规则：

- 由字母、数字、下划线组成。
- 不能以数字开头。
- 不能使用关键字，如 `int`、`class`、`return`。

好名字能救命。`sum` 比 `s` 更清楚，`studentCount` 比 `x1` 更像人写的。

## 3. 基本数据类型

| 类型 | 用途 |
| --- | --- |
| `bool` | 真/假 |
| `char` | 单个字符 |
| `int` | 整数 |
| `float` | 单精度小数 |
| `double` | 双精度小数 |
| `string` | 字符串，需要 `<string>` |

```cpp
int age = 18;
double score = 91.5;
char level = 'A';
bool passed = true;
```

## 4. 变量、常量与初始化

变量是能改的盒子，常量是贴了“别动我”标签的盒子。

```cpp
int n = 10;
const double PI = 3.1415926;
```

推荐使用初始化，不要让变量“裸奔”：

```cpp
int x{};       // 初始化为 0
double y{3.5};
```

## 5. 运算符与表达式

| 类别 | 运算符 |
| --- | --- |
| 算术 | `+ - * / %` |
| 关系 | `> < >= <= == !=` |
| 逻辑 | `&& || !` |
| 赋值 | `= += -= *= /=` |
| 自增自减 | `++ --` |

注意：`=` 是赋值，`==` 是判断相等。写错以后，程序通常不会大喊“你错了”，它会默默跑偏。

## 6. 类型转换

自动转换：小范围类型可能转成大范围类型。

```cpp
int a = 3;
double b = a; // 3.0
```

强制转换：

```cpp
double x = 5.0 / 2;
int y = static_cast<int>(x);
```

整数除法要小心：

```cpp
cout << 5 / 2;   // 2
cout << 5.0 / 2; // 2.5
```

## 7. 输入输出

```cpp
int age;
cin >> age;
cout << "age = " << age << endl;
```

格式控制常用 `<iomanip>`：

```cpp
cout << fixed << setprecision(2) << 3.14159;
```

## 8. 常见坑

- 未初始化变量：里面可能是随机值。
- 整数除法：`1 / 2` 是 `0`。
- 字符和字符串：`'A'` 是字符，`"A"` 是字符串。
- 浮点数比较：不要直接用 `a == b` 判断小数是否相等。

## 9. 本章小结

- C++ 程序从 `main` 开始。
- 类型决定数据怎么存、怎么解释。
- 初始化是好习惯，未初始化是事故预告。
- 输入输出是程序和用户交流的窗口。
- 运算符优先级拿不准就加括号，括号不收费。
