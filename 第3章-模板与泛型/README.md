## 1. 函数模板

> 🔨 待补充详细内容和代码示例

### 概念
- 函数模板是一种通用函数的写法，使用类型参数
- 通过模板，可以编写与类型无关的代码

### 基本语法
```cpp
template<typename T>
T max(T a, T b) {
    return a > b ? a : b;
}
```
