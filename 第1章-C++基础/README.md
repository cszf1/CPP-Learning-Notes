## 1. 数据类型

数据类型是在告诉电脑：“这块内存打算放什么东西。”`int` 像整数盒子，`double` 像能装小数的盒子，`char` 像一个字符小抽屉。

| 类型 | 常见用途 |
| --- | --- |
| `int` | 整数，如人数、下标 |
| `double` | 小数，如成绩、距离 |
| `char` | 单个字符 |
| `bool` | 真或假 |
| `string` | 字符串，需要 `#include <string>` |

```cpp
int age = 18;
double score = 92.5;
char grade = 'A';
bool passed = true;
string name = "Zhang";
```

注意：`'A'` 是字符，`"A"` 是字符串。一个是单人座，一个是小车厢。

## 2. 变量与常量

变量可以改，常量不该改。

```cpp
int count = 0;
count = count + 1;

const double PI = 3.1415926;
```

推荐用 `const`，少用 `#define` 定义常量。`const` 更像有身份证的变量，编译器知道它的类型；`#define` 更像文字替换，粗暴但不够讲究。

## 3. 运算符与表达式

| 类型 | 示例 | 说明 |
| --- | --- | --- |
| 算术 | `+ - * / %` | `%` 只能用于整数取余 |
| 比较 | `> < == !=` | `==` 判断相等，别写成赋值 `=` |
| 逻辑 | `&& || !` | 与、或、非 |
| 赋值 | `= += -=` | `x += 2` 等价于 `x = x + 2` |

```cpp
if (score >= 60 && score <= 100) {
    cout << "成绩有效";
}
```

## 4. 分支与循环

`if` 负责做选择，`for/while` 负责重复劳动。电脑最擅长重复劳动，不会抱怨，也不会摸鱼。

```cpp
if (age >= 18) {
    cout << "成年人";
} else {
    cout << "未成年人";
}

for (int i = 0; i < 5; ++i) {
    cout << i << endl;
}
```

`for` 适合次数明确，`while` 适合“条件满足就继续”。

## 5. 数组、指针与引用

数组是一排同类型格子：

```cpp
int a[3] = {1, 2, 3};
cout << a[0];
```

指针保存地址，引用是别名。

```cpp
int x = 10;
int* p = &x;
int& r = x;

*p = 20;
r = 30;
```

类比一下：变量是房间，指针是门牌号，引用是这个房间的外号。外号不能换房间，门牌号可以指向别的房间。

## 6. 函数基础

函数就是把一段功能装进盒子，需要时拿出来用。

```cpp
int add(int a, int b) {
    return a + b;
}
```

| 写法 | 特点 |
| --- | --- |
| 值传递 `int x` | 传副本，安全但改不到原变量 |
| 引用传递 `int& x` | 能改原变量 |
| 常量引用 `const string& s` | 不复制大对象，也不允许修改 |

## 7. 本章小结

- 类型决定内存怎么解释。
- 变量能改，`const` 常量不该改。
- `==` 是比较，`=` 是赋值，别让它们“串门”。
- 数组从 0 开始，下标越界是初学者常见事故现场。
- 指针不神秘，它只是地址；真正要小心的是空指针和野指针。
